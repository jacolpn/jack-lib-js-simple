# jack-lib-js-simple
A simple library javascript

## Installing
Install the library by `npm install jack-lib-js-simple``.

## How to use
If you used npm, just import the file:
```js
  import Console from 'jack-lib-js-webpack-jest';
```

And now you can use it like:
```js
  Console(message)
```
